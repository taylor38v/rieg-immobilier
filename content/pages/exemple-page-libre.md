# Bienvenue sur cette page exemple

Cette page a été créée pour montrer comment ajouter une nouvelle page depuis le CMS. **Tu peux la supprimer ou la modifier** depuis l'interface `/admin/`.

## Comment créer une nouvelle page

1. Connecte-toi sur `/admin/`
2. Clique sur **Pages libres**
3. Clique sur **Nouvelle page**
4. Remplis le titre, le slug (l'URL), et le contenu
5. Publie — la page sera live dans 2-3 minutes

## Tu peux écrire en markdown

- Listes à puces
- **Texte en gras** ou *en italique*
- [Liens vers d'autres pages](/contact/)
- Citations :

> Ceci est une citation. Pratique pour mettre en avant une phrase forte.

## Et même intégrer des images

Upload une image via le CMS, elle sera disponible dans `/uploads/`.
